§1 What this gate found

The gate did its job: probing the live origin surfaced a real perimeter defect that the unit tests didn't — every per-matter route resolved the vault before checking the Access token, leaking an unauthenticated 404/400 oracle that let a caller distinguish existent from nonexistent matters. Fixed in PR #16 (auth now resolves first on all 19 routes; regression test encodes the no-oracle invariant), redeployed, and re-verified below (item 2).
